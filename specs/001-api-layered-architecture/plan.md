# Implementation Plan: API Layered Architecture

**Branch**: `001-api-layered-architecture` | **Date**: 2026-04-13 | **Spec**: [spec.md](./spec.md)

## Summary

Split `api/routes.py` (1,872 lines, 35 endpoints) into domain-scoped routers and introduce an orchestration service layer. HTTP handlers become thin pass-throughs; business logic moves into injectable service classes that delegate to existing `services/` and `crud/` layers. Zero behaviour change — all URLs, request shapes, and response shapes are preserved.

## Technical Context

**Language/Version**: Python 3.11
**Primary Dependencies**: FastAPI, SQLAlchemy async, Pydantic v2
**Storage**: PostgreSQL (via existing `crud/` layer and `AsyncSession`)
**Testing**: pytest + pytest-asyncio, existing `TestClient` fixtures
**Target Platform**: Linux (k3s)
**Project Type**: Web service (API layer refactor only)
**Performance Goals**: No change — refactor must not degrade response times
**Constraints**: All 35 endpoint URLs unchanged; all 1,100+ tests pass without modification
**Scale/Scope**: 35 endpoints → 6 domain routers + 7 orchestration services

## Phase 1: Service Interface Contracts

```python
class PetImageService:
    async def get_or_generate(repo_owner, repo_name, stage, session, storage) -> bytes | None

class AnimatedGifService:
    async def get_or_generate(repo_owner, repo_name, stage, pet, session, storage) -> bytes

class PetResurrectionService:
    async def resurrect(pet, session, now) -> Pet  # raises ResurrectionNotAllowed

class WebhookProcessorService:
    async def process(payload, signature, event_type, session) -> dict  # raises InvalidSignature

class ContributorBadgeService:
    async def get_badge_svg(repo_owner, repo_name, username, github_service) -> bytes

class LeaderboardService:
    async def get_leaderboard(session) -> list[dict]

class PetAdminService:
    async def get_settings(pet, session) -> dict
    async def update_settings(pet, data, session) -> dict
    async def exclude_contributor(pet, github_login, session) -> None
    async def remove_exclusion(pet, github_login, session) -> None
    async def reset_pet(pet, session) -> Pet
```
